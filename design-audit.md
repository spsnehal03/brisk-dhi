### Reusable Component Patterns

| Component | Pattern |
|-----------|---------|
| **Header / Nav** | Sticky flex container with logo left, nav links right; hover underline using `::after` animation. |
| **Button** | `.btn` base – flex, centered, gap, padding, border‑radius, transition on background, color, border, transform, shadow. Primary/Secondary variants via color variables. |
| **Card** (service, stat, gallery) | Background + border + radius + padding; hover → elevation (box‑shadow) + translateY / scale. |
| **Grid Sections** | Section wrapper uses `padding-block: var(--section‑padding‑y)`; inner grid defined per section (2‑col, 4‑col, 3×2). |
| **Image placeholders** | Fixed aspect‑ratio, gradient background, border, radius, centered placeholder text via `::before`. |
| **Form fields** | Full‑width input, border, radius, focus outline using `--color‑accent`. |
